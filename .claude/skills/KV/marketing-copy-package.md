## Feature Highlights

- **Automatic routing** — Every input is classified by domain and risk before anyone touches it.
- **Risk tiers on everything** — Tier 0 runs free on local models. Tier 3 stops the system and calls the CEO.
- **MasterPlanner gate** — Large tasks produce a plan and wait for your approval before spending tokens.
- **Built-in contrarian** — Every investment recommendation gets a mandatory bear case from an independent analyst.
- **Self-maintaining** — The Custodian audits all 175 agent prompts for drift, compresses cache, and reports what is stale.
- **Six compliance frameworks** — COSO, SOC 2, NIST CSF, SOX, COBIT, CIS. Real controls, not decoration.
- **Free-tier routing** — Trivial tasks go to local Ollama models. The system does not waste money on easy work.
